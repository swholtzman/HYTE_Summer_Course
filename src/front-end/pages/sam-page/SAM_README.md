# sam-page — your example page (and your first task!)

This is the **simplest possible page**: just one HTML file and one CSS file, and
**no JavaScript**. It's here for you to copy.

- `sam-page.html` — the content (headings, text, a list, a link)
- `sam-page.css` — styles just for this page

---

## ⭐ Intro task: make your own page

Follow these steps to add **your own** page to the project.

### 1. Copy this folder

Copy the whole `sam-page` folder and rename the copy to your first name + `-page`.
For example, if your name is Alex: **`alex-page/`**

### 2. Rename the files inside

Inside your new folder, rename the two files to match:

- `sam-page.html` → `alex-page.html`
- `sam-page.css` → `alex-page.css`

### 3. Fix the CSS link

Open your `alex-page.html` and update this line so it points at YOUR css file:

```html
<link rel="stylesheet" href="alex-page.css" />
```

(Leave the `global.css` line exactly as it is.)

### 4. Make it yours

Change the parts marked with `👉` in the HTML (your name, your sentence, your
list) and try changing a colour in the CSS. Refresh your browser to see it.

### 5. Add a link on the home page (bonus)

In `front-end/pages/main-page/main-page.html`, copy the example link line and
change it to point at your page so people can find it.

### 6. Save it with git

In the terminal, from the project folder:

```
git add .
git commit -m "Add alex-page"
git push
```

> 💡 Your instructor may ask you to use a **branch** instead of pushing straight
> to main — ask if you're not sure.

That's it — you've created and shared your first web page! 🎉
