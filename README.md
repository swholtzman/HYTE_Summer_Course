# HYTE_Summer_Course
Basic HTML/CSS/JavaScript "app" for helping guide HYTE students in creating simple projects.

Goal:
- Provide HYTE students with a reference for how they could potentially set up their projects
- Simple html/css/javascript intro content
- Local storage for login page
- Optional deeper integration with the profile page form submitting to a database endpoint

Must Use:
- clear separation of front-end and back-end functionality
- SOLID and DRY coding mechanisms 
- use shared "common" and "utils" folders for functions that are shared between components
- must use global.css for any styling thats used among multiple pages (headers, fonts, colours, sizing, borders, padding, etc)
- must use scoped per-component/per-page .css files for styles that are scoped to one component, page, form, etc
